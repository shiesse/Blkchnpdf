10. Документация функций смарт-контракта
 AddShop: Добавление нового магазина.
 AddItem: Добавление товара в магазин.
 BuyItem: Совершение покупки.
 RateItem: Оценка товара.
 RefundItemRequest: Запрос на возврат товара.
